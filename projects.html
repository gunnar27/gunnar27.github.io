Title of the project:

Pacmen Game

Description of the project:

The Pacmen game gives the user the ability to create as many PacMen figures as they like and make them run randomly over the screen!

How to Run:

Download all files and open the index.html file in your browser! Click 'add PacMan' as many times as you'd like then click 'Start Game.'

Roadmap of future improvements:

In the future, i would like to add different character choices into the game!

Title of the project:

Real Time Bus Tracker

Description of the project:

project displays a map of Boston using Mapbox. The project also pulls data from the Boston MTA live API to place markers on bus stops and displays a pop up with the associated latitude and longitude positions.

How to Run:

will need to create your own Mapbox key and then download all files to your computer to run!

Roadmap of future improvements:

the future, I would like to add further functionality to show subway lines!
